Features

Multi-format

Supports Claude Code, Cursor, and Codex CLI transcripts. Auto-detected.

Self-contained

Single HTML file with no external dependencies. Email it, host it, embed it.

Web Editor

Built-in editor to browse sessions, edit prompts, exclude turns, and add bookmarks.

Interactive Player

Playback with speed control, keyboard shortcuts, diff views, and chapter navigation.

Themes

6 built-in themes plus custom theme support with full CSS override.

Docker

Run in a container with read-only session mounts for sandboxed usage.

Privacy

Everything runs locally. Automatic secret redaction for API keys, tokens, and secrets.
